Automated backups

A tool for all your back-ups to an internal or external drive!

Keep your Odoo data safe with this module. Take automated back-ups and write them to an external server. You can even specify how long local backups and external backups should be kept, automatically!

Test connection

Checks your credentials in one click

Want to make sure if the connection details are correct and if Odoo can automatically write them to the remote server? Simply click on the 'Test Connection' button and you will get message telling you if everything is OK, or what is wrong!
